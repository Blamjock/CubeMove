# CubeMove
Videogame Eventhorizont Project 1 Gennaio  2016

Concept:
Creazione di un gioco che attinga da meccaniche FPS (sparo, salto, camera in prima persona) ma si sviluppi in puzzle di navigazione spaziale, rimuovendo il conflitto tipico del genere.

Estetica:
Le estetiche alla base del gioco sono la claustrofobia, evocata tramite spazi angusti, simmetrie e giochi di luce; il disorientamento dato dallâ€™assenza di landmarks e da HUD minimale (assenza di minimap o tracker spaziali, hud diegetico).
